# data-dashboard-streamlit
Interactive data dashboard built with Streamlit to visualize business metrics and insights.


## Overview
This project builds an interactive data dashboard using Streamlit to visualize and explore business metrics such as revenue, sales trends, and performance insights.

## What I Did
- Created a dataset representing business transactions
- Built an interactive dashboard using Streamlit
- Visualized key metrics using charts and filters
- Enabled dynamic exploration of data

## Tools Used
- Python
- Pandas
- Streamlit
- Matplotlib

## Key Insights
- Dashboards allow quick understanding of business performance
- Interactive filters help explore data efficiently
- Visualization improves decision-making
